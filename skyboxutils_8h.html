Detailed Description

Utility functions for loading and applying a cubemap-based skybox in VTK.

This header provides functions to load six images as a cubemap texture and apply it as a skybox to a VTK renderer. It supports .png, .jpg, and other formats recognized by VTK's image reader factory.

vtkSmartPointer< vtkOpenGLTexture > LoadCubemapTexture ( const std::vector< std::string > &  faceFilenames)

Loads six images and creates a cubemap texture for skybox rendering.

The filenames should be passed in this order:

  • Positive X (px)
  • Negative X (nx)
  • Positive Y (py)
  • Negative Y (ny)
  • Positive Z (pz)
  • Negative Z (nz)
Parameters
faceFilenamesA vector of 6 file paths representing cubemap faces.
Returns
vtkSmartPointer to the generated cubemap texture.

Loads six images and creates a cubemap texture for skybox rendering.

The images must be ordered as: +X, -X, +Y, -Y, +Z, -Z (i.e., px, nx, py, ny, pz, nz). Each image is read, flipped along the Y-axis if needed, and assigned to the appropriate face.

Parameters
faceFilenamesA vector of six image file paths.
Returns
vtkSmartPointer<vtkOpenGLTexture> A cubemap texture with all six faces assigned.
