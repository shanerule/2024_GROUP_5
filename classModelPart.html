Detailed Description

Encapsulates a single part in the model tree, including its hierarchy, display properties, and VTK pipeline.

Manages child items, display data (name, visibility, colour), STL file loading, filter application (clip, shrink), and provides actors for on‐screen and VR rendering. :contentReference[oaicite:0]{index=0}:contentReference[oaicite:1]{index=1}
